# 🚀 C++ Practice Exercises (Beginner → Advanced)

A structured set of C++ programming exercises designed to build strong fundamentals step by step — from basic variables to loops and logic building.

---

## 📌 How to Use This Practice Sheet

- Solve problems in order: **Easy → Medium → Hard**
- Write each solution in a separate `.cpp` file
- Focus on logic clarity and clean code
- Revisit hard problems after completing the full set

---

# 📘 1. Variables & Data Types

## 🟢 Easy

- Create variables for name, age, and city and print them
- Store two numbers and print their sum
- Store a character and display it
- Create a boolean variable and print it
- Store a float value and display it

## 🟡 Medium

- Take name and age from user and display a sentence
- Store marks of 3 subjects and calculate total
- Swap two numbers using a temporary variable
- Store salary (double) and print yearly salary
- Input student details and print formatted output

## 🔴 Hard

- Input 5 subject marks and calculate percentage
- Use `long long` for large number multiplication
- Build a student report card system
- Apply GST on product price
- Create a simple ID card system
